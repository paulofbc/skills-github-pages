# pfbc-blog

Personal blog and knowledge base — short, practical notes on the
technical things I'm learning. Hosted on GitHub Pages with Jekyll
(Hydejack theme).

Live site: <https://paulofbc.github.io>

## Local preview

```sh
bundle install
bundle exec jekyll serve --drafts
```

See [`CLAUDE.md`](./CLAUDE.md) for conventions (post format, categories,
permalink scheme).
